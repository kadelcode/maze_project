The 3D Maze Project

A 3D computer graphics game using raycasting; a technique for rendering 3D objects.

Features

3D Graphics

The maze is rendered in 3D. This feature creates a realistic and immersive experience.

Map

This feature shows the player where they are in the maze, and it can also show the player the different paths that they can take.

Weapons & Enemies

This feature makes the game more adventurous

About

Project Inspiration

The main inspiration of this project is from my love of old 3D computer games such as: virtual cops, GTA vice-city. Based on the movie/real-world experience, which made 3D games more interesting and realistic; I felt this would be an opportunity for me to have an idea on how they are created and also getting familiar with the technologies.
